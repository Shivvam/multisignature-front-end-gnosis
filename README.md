# Front End for Gnosis Based Multi Signature Wallet

## That lets

- Create a wallet
- Add Owners
- Remove Owners
- Chnage Threshold
- Transact (By all Owners)


## Tech Stack Used

- Next JS App [Front End]
- Tailwind CSS [For UI Styling]
- Redux Toolkit [For State Management ]
- JWT [ For Session management with Supabase] 
- Supabase [For Database, to store address => [“safe1”,”safe2”...]
- Ethereum Javascript API [Interacting with local or remote ethereum node using HTTP, IPC or WebSocket]
